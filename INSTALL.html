F8 Lite Theme Instructions

About this theme

F8 lite is a free photography and multimedia portfolio theme for Wordpress. The theme design was developed with photographers in mind who want a minimal layout, with their photos and multimedia featured front and center. But, the theme has been used to create everything from e-commerce sites, book review sites, and everything in between. In the future, we will be releasing "child themes" to provide design alternative and additional applications (apps) that will work with F8 Lite. You'll definitely want to subscribe to GraphPaperPress.com to get all the latest developments and enhancements for F8 Lite.

Installing this theme

  1. Unzip the folder contained in the archive to your computer
  2. Upload the folder and its contents to the themes directory of your blog, located in "wp-content/themes" where WordPress is installed.
  3. Click on the Design tab from the WP admin panel, then select the theme to activate.
  4. Click on View Site at the top of the page.

Initial Setup

F8 Lite is a 950px wide, one column theme. The archive and front page layout is built on a three column grid (310px wide with 10px gutters). We need to set up your default image and thumbnail sizes to allow for easy posting and consistent design structure. Click Settings - Miscellaneous and set your thumbnail size to 310px wide by 150px tall. Set your medium size images to 950px wide and 700px tall. For post thumbnails, paste in the url to the 310px wide X 150 px tall thumbnail into the custom field, with key value "thumbnail". For slideshow images, paste in the url to the 950px wide medium size image into the custom field, with key value "image". By default, the homepage slideshow shows only those posts with a custom field "image." The slideshow shows the most recent 4 posts in category 1. Widgets

The bottom three column area is 100% widgetized, which you can control by clicking Design - Widgets, from within your Wordpress admin panel. You can add and delete widgets by clicking Design - Widgets, from within your Wordpress admin panel.

Multimedia Custom Fields

You can also add multimedia, including video, Google Maps, and virtually any type of embeddable media by entering "multimedia" as the key value and paste your YouTube, Vimeo, or any other html embed code into the value field. You can read more about using custom fields here.
